## 建议的备用方案：

### 方法1：使用 Google 图片搜索
搜索 "[品牌名] logo png" 或 "[品牌名] logo svg"，下载高质量版本

### 方法2：访问官网
大部分品牌的官网都有 logo 下载或新闻资料包

### 方法3：临时使用文字
如果暂时找不到 logo，应用会显示默认的图标

## 文件命名规则：
- 文件名必须全小写
- 空格用下划线替换
- 保持原始格式（svg 或 png）
